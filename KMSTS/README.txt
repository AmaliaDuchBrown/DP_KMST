This folder contains the following subfolders:

1) edgesPreProcess: contains programs to transform graphs from 
repositories to entries for programs in 3) and 4)

2) small-exemples: contains toy examples for programs in 3) and 4)

3) KMST_VSuperF: contains go programs of kmsts version super filters 
with and without lock.

4) KMST_VSingleF: contains go programs of kmsts sequential and single 
root with and without lock.

The programs in 3) and 4) are almost ready to use, they only need to be compile with the go command of the go programming language.
